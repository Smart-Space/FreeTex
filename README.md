# FreeTex - 免费的智能公式识别神器


## 🌟 简介

FreeTex是一个免费的公式智能识别软件
特色：

- 用本地资源 不上网也能用

- 自适应显卡加速    
  急速识别无需等待

- 支持图像粘贴，识别一键解决

- 多类型图像识别

- 导出结果支持word/latex

## 🚀 鸣谢
